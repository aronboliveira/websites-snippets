# Instructions for Google Chrome

- pt-BR:
    1. Abra o Gerenciador de Favoritos ((Clique nos três pontos no limite da janela, à direita da barra de URL -> Favoritos e Listas -> Gerenciador de Favoritos) ou Crtl + Shift + O)
    2. Clique com o botão direito do mouse em uma área de plano de fundo do gerenciador aberto
    3. No modal aberto, clique em "Adicionar novo favorito" (sugestão: inclua toda as funções em uma pasta única aparecendo no diretório da Barra de Favoritos);
    4. Preencha o formulário: O nome é sua escolha, enquanto a URL deve ser a função (completa, basta copiar e colar a sua linha) em versão minificada e executável (encontrada em arquivos com sufixo especificado, como whatsapp_min.js);
    5. Clique no botão "Salvar";
    6. Agora toda vez que quiser executar a função, basta clicar no favorito - como faria com a navegação para um site!
